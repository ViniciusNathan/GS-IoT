## Objetivo do Projeto
Este projeto tem como finalidade promover *saúde, bem-estar e produtividade* em ambientes de estudo e trabalho. Utiliza um Arduino UNO para criar um *Timer Inteligente de Foco*, baseado no método Pomodoro, incentivando pausas saudáveis e melhorando a eficiência durante atividades cognitivas. Pomodoro indica tempos de 25min de foco em estudos.

##  Contexto – Futuro do Trabalho
No cenário atual de trabalho híbrido e estudo remoto, longos períodos sem descanso prejudicam:
•⁠  ⁠concentração,
•⁠  ⁠saúde mental,
•⁠  ⁠desempenho,
•⁠  ⁠e equilíbrio entre foco e descanso.

Nosso projeto contribui para o conceito de *Tecnologias Humanizadas*, alinhado ao tema do Global Solution, oferecendo uma solução acessível e funcional.

##  Funcionalidades
•⁠  ⁠Inicia o ciclo de foco ao pressionar o botão.
•⁠  ⁠Exibe o tempo restante no display TM1637.
•⁠  ⁠Resetar o tempo ao clicar no botão
•⁠  ⁠Promove pausas regulares para reduzir estresse e fadiga mental.

##  Componentes Utilizados
•⁠  ⁠Arduino UNO  
•⁠  ⁠Display TM1637 (4 dígitos)  
•⁠  ⁠Botão (start/reset)  

##  Funcionamento
1.⁠ ⁠O usuário pressiona o botão.  
2.⁠ ⁠O temporizador inicia a contagem regressiva.  
3.⁠ ⁠O display mostra o tempo em formato MM:SS.  
4.⁠ ⁠Ao final, o sistema alerta o usuário.  
5.⁠ ⁠O ciclo é reiniciado.
